Roadmap

研究路线：从理论到能跑实验、能发论文

如果你是博士生或刚开始做 Agentic RL 研究，这是一份按时间线组织的学习和研究路线。 每个阶段附带必读论文、推荐代码库和验证 milestone。

Phase 1: 理论基础（第 1-2 周）
RL 基础理论

必读：Sutton & Barto Ch. 13 (Policy Gradient Methods)
推荐：Schulman 2015 (TRPO), Schulman 2017 (PPO) 原始论文
博客：Lilian Weng "Policy Gradient Algorithms"
Milestone：能手推 Policy Gradient Theorem 和 PPO clipping objective。

动手实验

用 TRL 的 PPO trainer 训练一个小 LLM（如 Qwen2.5-0.5B）做简单数学题。 目标不是做出好结果，而是理解 rollout → reward → advantage → update 的完整流程。
推荐代码：TRL (HuggingFace)

Phase 2: LLM 对齐 RL（第 3-4 周）
RLHF Pipeline

必读：InstructGPT (Ouyang et al. 2022) — RLHF 的工业化实践
必读：DPO (Rafailov et al. 2023) — 理解 offline vs online RL 的 tradeoff
推荐：Ziegler et al. 2019 "Fine-Tuning LMs from Human Preferences"
Milestone：能画出 SFT → RM → PPO → DPO 的完整技术关系图。

深度理解

DPO = RL 的 closed-form 解？理解其 Bradley-Terry assumption。
Preference 数据 vs Verifiable reward 的区别和联系。
博客：HuggingFace "RLHF: Aligning LLMs"

Phase 3: Reasoning RL（第 5-8 周）
核心论文精读

精读（逐节、逐公式）：
1. DeepSeekMath (Shao et al. 2024) — GRPO 完整理解
2. DeepSeek-R1 (2025.01) — RL for reasoning 的 milestone
3. DAPO (2025.03) — 算法-系统 co-design
4. Open Reasoner Zero (2025) — 可复现 baseline
Milestone：能复述 GRPO 的完整 pipeline 并指出和 PPO 的 3 个关键差异。

复现实验

用 veRL 或 OpenRLHF 复现 GRPO 数学推理实验。
目标 benchmark：MATH-500 或 GSM8K。
验证：SFT baseline → GRPO training → 观察 reward/KL/entropy 曲线 → eval pass@1 是否提升。
推荐代码：veRL, OpenRLHF

Phase 4: Agent RL 前沿（第 9-12 周）
Agent RL 论文

精读：Agent Q (2024), RAGEN (2025), Search-R1 (2025)
泛读：ReST-MCTS* (2024), rStar-Math (2025), Absolute Zero (2025)
PRM 方向：Math-Shepherd (2024), PRIME (2024), OmegaPRM (2024)
Milestone：能画出"Agent 轨迹如何变成 RL 训练信号"的完整 pipeline。

确定研究方向

候选方向（按理论/实验偏好选择）：
A. Credit Assignment：理论创新，分析/改进多步 agent RL 的信用分配
B. Reward Design：PRM 构建、reward composition、verifier robustness
C. Training System：效率、稳定性、可扩展性的系统研究
D. New Domains：将 RL 训练扩展到新 agent 领域（web, robotics, science）
E. Theory：sample complexity、convergence guarantees、exploration theory

推荐代码仓库
veRL

字节跳动出品的大规模 RL 训练框架。支持 FSDP/Megatron + Ray，适合做大规模 reasoning RL 实验。如果你的实验室有 64+ GPU，首选。

OpenRLHF

社区驱动的 RLHF 框架，集成 vLLM 加速 rollout。算法覆盖全（PPO, GRPO, RLOO, DPO, KTO），适合中等规模实验。

TRL

HuggingFace 官方出品，集成 transformers 生态，最容易上手。适合快速原型验证和教学。

Open Reasoner Zero

可复现的 R1-Zero 训练 baseline。代码 clean，适合作为论文实验的 starting point。

推荐关注的研究组
  • DeepSeek：GRPO, R1, DeepSeekMath — reasoning RL 的核心推动者
  • ByteDance Seed：DAPO, veRL — 大规模 RL 系统
  • Qwen (Alibaba)：Qwen 系列模型的 RL post-training 经验
  • Allen AI (Ai2)：Open RL 研究，Tülu 系列
  • Meta FAIR：Llama RL post-training, RLOO 相关工作
  • UC Berkeley (BAIR)：Agent Q, RL theory
  • THU NLP (清华)：Agent 评测和 benchmark
